
# 🚀 AI-Powered Code Reviewer and Quality Assistant

## 📖 Overview
This project is an AI-powered code analysis tool designed to improve **code quality, readability, and documentation**.  
It performs:
- Function-level analysis  
- Error detection  
- Visualization  
- Automated docstring generation  

---

## 🛠️ Tools, Libraries & Technologies Used

### 💻 Programming Language
- Python  

### ⚙️ Core Technologies
- Abstract Syntax Tree (AST) – for parsing and analyzing code structure  
- LLM APIs – for AI-based docstring generation  

### 📚 Libraries
- ast – parsing Python code  
- pandas – data handling and structuring  
- matplotlib / plotly – visualization (graphs & charts)  
- json – structured data representation  

### 🌐 Frameworks / Platforms
- Streamlit / Flask – UI dashboard  
- REST APIs – LLM integration  

### 🧪 Testing & Validation
- pytest – unit testing  
- Coverage tools  

---

## ✨ Features
- 📂 Upload file/folder for analysis  
- 🔍 Function-level code analysis  
- ⚠️ Error detection (PEP257 violations)  
- 📊 Graph insights (complexity vs functions)  
- 📝 AI-generated docstrings  
- 📈 Dashboard with filtering & reports

## 🔄 Development Process

### 1️⃣ Requirement Analysis
- Identified need for code quality + documentation tool  
- Defined modules: analysis, visualization, docstrings  

### 2️⃣ Design Phase
- Designed system workflow:  
  **Input → Parsing → Analysis → Output**  
- Created UI structure (dashboard & views)  

### 3️⃣ Implementation

#### 🔹 Milestone 1: Function Analysis
- Parsed code using AST  
- Extracted function details  
- Displayed results in table & JSON  

#### 🔹 Milestone 2: Code Quality
- Detected errors and violations  
- Provided fix suggestions  
- Added graph insights  

#### 🔹 Milestone 3: Docstring Assistant
- Integrated LLM  
- Generated docstrings in multiple styles  
- Compared and updated documentation  

#### 🔹 Milestone 4: Dashboard & Testing
- Built interactive dashboard  
- Added filters, search, export  
- Implemented testing and validation
